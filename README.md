# Laravel + Docker 

Тестовое задание: Необходимо было создать страницу авторизации/регистрации (имя, почта, телефон) и с небольшим личным кабинетом и просмотром данных профиля + возможностью изменить данные профиля. 

## Инструкция по запуску 

Проект запускается командой: docker-compose up -d 

## Навигация 

Вся навигация происходит через меню 

Адреса основных страниц по ТЗ: 
Главная страница с возможность авторизоваться или зарегистрироваться: http://localhost:8080 
Регистрация: http://localhost:8080/register 
Авторизация: http://localhost:8080/login
Дашборд (личный кабинет): http://localhost:8080/dashboard
Профиль: http://localhost:8080/profile 
Изменение профиля: http://localhost:8080/profile/edit
